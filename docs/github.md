# GitHub

- Repository: https://github.com/ilestri/Logdeck
- Clone: `git clone https://github.com/ilestri/Logdeck.git`

This repository is kept source-run focused. Clone it, run `swift run Logdeck`, and use `swift build` plus `swift test` for local verification.

GitHub Issues, Wiki, Projects, Actions, and public support intake are disabled. The repo is public for source access, not for running automation or a user-support queue.
